==========
Colordiffs
==========

Colors for your git diffs

How to run
==========

::

    git diff HEAD^ | python colordiffs.py | less -r

.. |screenshot| image:: https://github.com/ngzhian/colordiffs/raw/master/colordiffs.png
    :alt: colordiffs screenshot

Installation
============

::

    pip install colordiffs

Requirements
============

::

    Python 2.x
    Pygments
